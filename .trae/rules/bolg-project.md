# 技术选型规范:

- 全栈框架:Next
- 开发语言:TypeScript
- 包管理器:pnpm
- 状态管理库:zustand
- ui:tainwindcss，shadcn-ui
- 代码审查与规范:Eslint+prettier
- 动画库:GASP

# UI样式规范:

- 响应式布局:支持pc端，移动端适配
- 复杂动画交互使用GASP库
- 模块化架构:各个模块相互之间不要过分依赖
- 设计风格规范:1.简约不花哨。2.tap栏支持液态玻璃的效果。3.主体颜色不要采用渐变色，采用单一颜色:白色，黑色，米白色，米黄色等能体现高级感的颜色。
- 预留支持大模型的位置(现在不要实现):预留大模型的用处:1.辅助用户查找相关文章。2.辅助用户回答相关问题。预留大模型形式:1.tap栏的模块。
- 持续提升用户查看博客文章的体验

# 系统安全:

- 密钥存储安全:密钥，API key等使用环境变量的方式存储
- 前后端敏感信息隔离:前端只做页面展示，任何敏感信息都放在后端
- XSS过滤:前端输入的评论内容，必须在后端存储前进行转义
- CSP内容安全:禁止页面加载外部脚本
- 接口限流:评论接口和文章列表接口加限制，同一IP一分钟那最多可评论五次
